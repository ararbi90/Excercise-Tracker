# Excercise-Tracker
This is a crud application built on a MERN Stack.
<br/>
It supports the following operations:
<ol>
  <li>
    A User can be created.
  </li>
  <li>
    A user may enter excercises.
  </li>
  <li>
    A user my edit/delete an excercise
  </li>
</ol>
